# RU

# Основы алгоритмизации и реализации задач на различных языках программирования

Данный репозиторий содержит реализацию трёх алгоритмических задач на различных языках программирования.  
Каждое упражнение выполнено отдельно и реализовано на нескольких языках.

Все программы были разработаны и протестированы в онлайн-компиляторах, поэтому единая точка входа (`main`) для всего проекта отсутствует — каждый файл является самостоятельной программой.

---

## Структура репозитория

Репозиторий содержит три упражнения:

- `Ex1.*` — Упражнение 1  
- `Ex2.*` — Упражнение 2  
- `Ex3.*` — Упражнение 3  

Каждое упражнение реализовано на следующих языках программирования:

- C++
- C#
- Go
- Java
- JavaScript
- Kotlin
- Python
- Ruby
- Rust
- Swift
- Assembly (для отдельных задач)
- Haskell (для отдельных задач)

---

## Описание упражнений

### Упражнение 1

Задача направлена на реализацию базового алгоритма обработки входных данных, выполнение вычислений и вывод результата в соответствии с поставленным условием.

Программа:
- принимает входные данные;
- выполняет вычисления по заданному алгоритму;
- выводит результат в консоль.

---

### Упражнение 2

Задача предполагает реализацию алгоритма с использованием условных операторов и/или циклов.

Программа:
- считывает входные значения;
- обрабатывает их согласно условию;
- выводит итоговый результат.

---

### Упражнение 3

Задача направлена на закрепление навыков работы с логическими конструкциями и вычислениями.

Программа:
- получает входные данные;
- выполняет требуемые вычисления;
- выводит результат в стандартный поток вывода.

---

## Требования

Для запуска программ необходимо:

- Онлайн-компилятор соответствующего языка  
или  
- Установленная среда разработки и компилятор выбранного языка программирования.

Каждый файл компилируется и запускается отдельно.

---

## Как использовать

1. Выберите нужное упражнение (`Ex1`, `Ex2` или `Ex3`).
2. Выберите язык программирования.
3. Откройте соответствующий файл в онлайн-компиляторе или локальной среде разработки.
4. Скомпилируйте и запустите программу.
5. Введите входные данные (если требуется).
6. Получите результат выполнения программы.

# EN

# Fundamentals of Algorithms and Multi-Language Implementation

This repository contains implementations of three algorithmic tasks in multiple programming languages.  
Each exercise is implemented separately and independently.

All programs were developed and tested using online compilers. Therefore, there is no single global `main` file — each source file represents a standalone program.

---

## Repository Structure

The repository contains three exercises:

- `Ex1.*` — Exercise 1  
- `Ex2.*` — Exercise 2  
- `Ex3.*` — Exercise 3  

Each exercise is implemented in the following programming languages:

- C++
- C#
- Go
- Java
- JavaScript
- Kotlin
- Python
- Ruby
- Rust
- Swift
- Assembly (for selected tasks)
- Haskell (for selected tasks)

---

## Task Description

### Exercise 1

This task focuses on implementing a basic algorithm that processes input data, performs required calculations, and outputs the result.

The program:
- reads input data;
- performs computations according to the algorithm;
- prints the result.

---

### Exercise 2

This task involves implementing logic using conditional statements and/or loops.

The program:
- reads input values;
- processes them according to the given condition;
- outputs the final result.

---

### Exercise 3

This task reinforces skills in logical operations and algorithmic thinking.

The program:
- accepts input data;
- performs the required calculations;
- outputs the result to the standard output.

---

## Requirements

To run the programs, you need:

- An online compiler for the chosen programming language  
or  
- A local development environment with the appropriate compiler installed.

Each file must be compiled and executed separately.

---

## How to Use

1. Choose the required exercise (`Ex1`, `Ex2`, or `Ex3`).
2. Select the programming language version.
3. Open the corresponding file in an online compiler or local IDE.
4. Compile and run the program.
5. Enter input data if required.
6. Review the output result.
